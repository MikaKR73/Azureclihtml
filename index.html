Welcome to Azure CLI

About Azure CLI

The Azure Command-Line Interface (CLI) is a set of commands used to create and manage Azure resources. It is designed to get you working quickly with Azure, with an emphasis on automation.

Features

  • Create and manage Azure resources from the command line
  • Automate your workflows with scripts
  • Integrate with other tools and services

Getting Started

To get started with Azure CLI, you can install it on your local machine or use it directly from the Azure Cloud Shell.

For more information, visit the official documentation.
